Тележный бот, предлагающий к просмотру случайный фильм из сформированной библиотеки в формате: название, страны создания, режиссер, оценка на кинопоиске, синопсис, постер
Бот делится на две части: сам бот (KinoBot.py) и парсеры кинопоиска. Kinoparser_primary - формирует список фильмов в директории и записывает к каждому фильму первоначальную информацию. KinoParser_secondary - обрабатывает уже каждый фильм - ищет режиссера, синопсис и т.д. KinoParser_Cleaner - нужен для очистки незагрузившихся фильмов (по неизвестной причине, на пару сотен распарсенных фильмов, несколько штук имеют пустые постер и описание, было решено просто удалять их из библиотеки)
p.s. возможно, кинопоиск меняет html-код страниц для избежания распарсивания, так что может понадобиться заново заходить в исходный код кинопоиска и смотреть название блоков
